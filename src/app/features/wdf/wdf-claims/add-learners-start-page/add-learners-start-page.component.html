Add learners Now you need to add learners to your claim

There needs to be a staff record in ASC-WDS for each learner that you want to add.

Staff records will need to include a name that matches the one shown on their learning award certificate, not an ID number, and a date of birth.

You'll need to know the amount of money you want to claim for each learner.

For each learner, you'll need to know:

  • their Unique Learner Number
  • their Candidate Registration Number
  • the amount of money you want to claim
  • whether the qualification is part of an apprenticeship
